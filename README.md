# Alen 1.0 - AI Sentiment Analysis Dashboard 🤖

This is a custom Natural Language Processing (NLP) model trained to analyze the sentiment of text and determine if it is positive, negative, or neutral. 

### Features
* **The Brain:** Powered by a custom-trained TensorFlow LSTM Neural Network.
* **The Interface:** An interactive web dashboard built entirely in Python using Streamlit.
* **The Output:** Displays immediate verdicts, confidence scores, and a visual sentiment gauge.

### How to Use
Just type a sentence into the text box and click "Ask Alen" to see the AI's thought process!
